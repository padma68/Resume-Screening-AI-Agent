Job Title: Backend Engineer – Machine Learning Platform

Company: NimbusTech Solutions
Location: Bengaluru, India (Hybrid)
Experience Required: 3-6 years

About the Role
We are looking for a Backend Engineer to join our ML Platform team. You will build and
maintain services that power our recommendation and data pipeline systems. You'll work
closely with data scientists to productionize models and with the platform team to keep
services reliable at scale.

Required Skills
- Strong proficiency in Python
- Experience building REST APIs (FastAPI, Flask, or Django REST Framework)
- Hands-on experience with SQL and at least one relational database (PostgreSQL/MySQL)
- Experience with Docker and containerized deployments
- Familiarity with cloud platforms (AWS or GCP)
- Understanding of machine learning model deployment / MLOps concepts
- Experience with Git and CI/CD pipelines
- Solid understanding of data structures and algorithms

Nice to Have
- Experience with Kubernetes
- Experience with message queues (Kafka, RabbitMQ, SQS)
- Exposure to PyTorch or scikit-learn
- Experience with monitoring/observability tools (Prometheus, Grafana)

Education
Bachelor's degree in Computer Science, Engineering, or a related field. Equivalent
practical experience is acceptable.

Responsibilities
- Design and build scalable backend services for ML-powered features
- Collaborate with data science to deploy and monitor models in production
- Write clean, well-tested, maintainable code
- Participate in code reviews and system design discussions
- Own services end-to-end, including on-call support
